Meeting Copilot — Perception → Planning → Action

Google DeepMind Enterprise Build Day
Agentic · Multimodal · Enterprise  |  Python · asyncio · FastAPI

Self-driving-style agent stack. Every layer is interchangeable behind two fixed schemas (Contract A / B); layers never call each other directly — they communicate through an in-process event bus (bus.py) over a shared Meeting State. Swap adapters or model backends via env config; the core is untouched.
